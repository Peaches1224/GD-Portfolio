Vector

Vector graphics are a type of image created using mathematical equations rather than pixels. They are composed of paths, which are defined by a start and end point, along with other points, curves, and angles. This makes vector graphics scalable to any size without losing quality, unlike raster graphics which can become pixelated when resized.
Key Characteristics of Vector Graphics: Scalability: Vector images can be resized infinitely without any loss of quality. File Size: Typically smaller file sizes compared to raster images, as they are defined by mathematical expressions rather than pixel data. Editability: Easier to edit individual elements within an image since each shape and line can be manipulated independently. Common Formats: SVG, AI (Adobe Illustrator), EPS, and PDF.
